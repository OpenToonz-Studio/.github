# OpenToonz Software — Portable Professional 2D Animation Studio

## About OpenToonz Software

OpenToonz Software provides a professional 2D animation suite originally developed by Studio Ghibli and used in productions including Princess Mononoke. OpenToonz Software combines raster and vector drawing, skeleton rigging, particle effects, motion tracking, scanning, coloring, and compositing. OpenToonz Software supports paper drawing scanning workflow, multi‑layer rendering, effects stack, camera controls, and lip syncing. OpenToonz Software integrates Xsheet (timeline), styles palette, glitch effects, and peer reviewed tools directly into the interface. OpenToonz Software enables independent animators to create studio‑quality 2D animation without licensing barriers. OpenToonz Software runs on portable storage, USB drives, and local user folders. OpenToonz Software is trusted by independent studios, anime creators, professional animators, students, and open source advocates worldwide.
